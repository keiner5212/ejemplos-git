## Quick Markdown cheatsheet

| You write | You get |
|-----------|---------|
| `# Title` | Big heading |
| `## Subtitle` | Smaller heading |
| `**bold**` / `*italic*` | **bold** / *italic* |
| `[link](https://example.com)` | Clickable link |
| `` `code` `` | Monospace inline |
| `- item` (one per line) | Bullet list |
| `1. step` | Numbered list |
| Blank line | New paragraph |
